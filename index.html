Dental Diagnosis Expert System

An expert system for dental disease diagnosis using machine learning.

Features
  • Diagnoses 6 dental diseases based on 28 symptoms
  • Uses K-Nearest Neighbors (KNN) algorithm
  • Interactive web interface built with Streamlit
  • Command-line interface option
Installation
# Clone the repository
git clone https://github.com/kavyareddy2115/dental-diagnosis-expert-system.git

# Navigate to the project directory
cd dental-diagnosis-expert-system

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run streamlit_app.py
Note: This is a static landing page. The full interactive application needs to be run locally.
